CMPT 350 Course Objectives

  • Students will understand HTML, CSS, and JavaScript as the base of the Web development
  • become familiar with the back-end (Java servers, NodeJS) and database storage
  • become familiar with the Ajax and Front-end Frameworks
  • become familiar with key concepts of Web servers and Web services
  • be briefly introduced to cloud computing and Web strategies
  • be briefly introduced to Web security
  • be briefly introduced to mobile Web
